#Obsidian 

# Obsidian同步的笔记
> 同步于ipad与 Mac

## Mac 
安装 Obsidian git插件



## Ipad

> 安装`ISH Shell`

app Store安装ISH


使用apk安装git
```shell
apk add git
```

git 一直卡住，一直拉不了远程仓库

ish里面可以使用mount 挂载命令 去指定挂载ipad的目录

后面在ipad安装working Copy，使用它去拉取远程仓库，但是吧这个东西copy到指定目录出来的东西是zip压缩包。。。

然后通过ish解压这个压缩包

这样就可以达到同步远程仓库的文件了。

但是这样真的太麻烦了。

后续找其他方法取代一下。
