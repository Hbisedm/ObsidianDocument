# TypeScript 内一些好用的类型的笔记

## Partial

> lib.es5.d.ts 文件的

`Partial<T>` 可以快速把某个接口类型中定义的属性变成可选的(Optional)

```typescript
interface People {
  age: number;
  name: string;
}
const p1: Person = {
  age: 23,
  name: "Sam",
};
type AnonymousPeople = Partial<People>;
const p2: AnonymousPeople = {
  age: 24,
};
```
